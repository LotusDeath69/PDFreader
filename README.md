# **PDF to speech** 

## **Usage** 
### Convert words in English from a pdf file into mp3 file(s) 

## **Requirements**
```
pyttsx3==2.90
PyPDF2==3.0.1
$ pip install -r requirements.txt
```

## **Parameters** 
### File: pdf file path 
### Size (default=50): Pages per audio file 
### Audio rate (default=200): Audio speed
